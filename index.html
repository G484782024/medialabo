プロジェクト課題について

世界(12ヶ所)の天気がわかるサイト。検索方法はテキストボックスではなく、都市名（国名）のプルダウンメニューによって検索することができます。また、天候、最高気温、湿度といった重要な情報が一目でわかるよう、色分けを行いました。このサイトの最も重要な情報である、国名、天候は文字だけの出力ではなく、GIF画像での出力をすることにより、一目でわかりやすくするだけでなく、動的な、シンプルすぎないUIとした。

各授業回の学習事項

第1回 WWW と HTML

HTMLは、要素によって構成されており、その中でのブロック要素、インライン要素に分類される。HTMLではコード上に入力しても、Webページ上では反映されない特殊文字が存在するため、<, >, ", ', &などは別の記述を使用する必要がある。

演習の成果(1): 演習1-3: HTML文書の作成
演習の成果(2): 演習1-4: HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

CSSは、 Webサイトのデザインを決める役割を果たしており、カラーや配置など、様々なデザインができる。また、CSSを書く際には、セレクタという対象要素を指定する記述が多くあり、一部または、条件に当てはまるものを変更したい場合は、セレクタを利用する。

演習の成果: 演習2-3: ウェブブラウザ説明ページの装飾

第3回 JavaScript の基本

JavaScriptは、Webページ上で複雑な機能を実現するプログラミング言語であり、目にみえるWebページ上の動作以外にも、コンソールの動きに変化を加えることができる。javaのように、for文やif文を使用できるが、違う点として、コンパイルを行う必要がないという点がある。実行に手間がかからないという利点があるが、デバッグを行うのが大変というデメリットもある。

演習の成果: 演習3-6: オブジェクトの参照

第4回 DOM と JavaScript

JavaScriptでは、前回行ったコンソール上での動作に加えて、Webページそのものを動作させることのできる言語である。その一例として、テキストボックス、ボタンを表示させ、クリックによって動作させることができる。JavaScriptでプログラミングを行う際には、エディタで先に記述するのではなく、Webページのコンソール上で動作を確認してからエディタに記述するのが、ミスの少ないプログラムを書くコツである。

演習の成果: 演習4-10: 足し算ページ

第5回 イベント処理

JavaScriptのDOMの機能の一部として、要素の新規作成、要素の編集、要素の削除が行える。要素の新規作成や削除の機能によって、HTMLファイルの記入を最小限に抑え、JavaScript上でデザインできる。また、要素の削除の機能によって、JavaScript特有のデザインの変更などが行えるようになる。

演習の成果: 演習5-7: DOMプログラミング

第6回 プロジェクト課題を完成させよう

学習内容としては、AJAX通信によるデータの要求、受取を行った。XMLHttpRequestというオブジェクトを使用して行う方法があるが、難易度が高いため、今回はaxiosというライブラリを使用して行った。また、プロジェクト課題では、idの入力による検索では、使い勝手が悪いと感じたため、プルダウンメニューによって、検索キーと国名を紐づけることによって、わかりやすいUIにした。
